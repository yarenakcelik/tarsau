# tarsau - Sistem Programlama Projesi

Bu proje, Linux/Unix ortamında C dili kullanılarak geliştirilen basit bir arşivleme programıdır.

Program `tar`, `rar` veya `zip` benzeri çalışır; ancak sıkıştırma işlemi yapmaz. Verilen metin dosyalarını tek bir `.sau` arşiv dosyası içinde birleştirir ve daha sonra bu arşiv dosyasını tekrar açarak dosyaları geri oluşturur.
